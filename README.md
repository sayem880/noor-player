# 🎵 Noor Player - Al-Quran Audio Player

A beautiful, mobile-friendly Quran audio player featuring **Mishary Rashid Alafasy** recitation with **Bengali translation audio**.

## 🌐 Live Player
[https://asym880.github.io/noor-player](https://asym880.github.io/noor-player)

## ✨ Features
- 🎙️ Mishary Rashid Alafasy recitation
- 📖 Bengali translation audio
- 💎 High quality / 📱 Low quality switching
- 🔁 Repeat modes (Off / One / All)
- 🌐 English / বাংলা UI toggle
- 📱 Background playback on Android
- ⏯️ Play from where you left off

## 🛠️ Tech Stack
- HTML5
- CSS3 (Mobile-first design)
- Vanilla JavaScript
- LocalStorage API
- GitHub Pages

## 📱 Browser Support
- Chrome (recommended for Android background playback)
- Firefox
- Safari
- Edge
